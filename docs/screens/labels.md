# Labels

Add and remove GCP labels on any instance.

## Usage

1. Select an instance from the dropdown.
2. Existing labels are shown as key/value pairs.
3. Add new labels with **Add label**.
4. Update labels by **Adding a label** with the same key.
5. Remove labels with the delete button next to each row.
6. Click **Save** to apply the changes.

## Label format

GCP label keys and values must:

- Contain only lowercase letters, digits, underscores, and hyphens
- Be at most 63 characters long
- Keys and values may contain lowercase letters, digits, underscores, and hyphens

## Notes

The Labels page can also be opened directly from the **Instances** page via the **Edit Labels** row action.
